Gambar Alat yang berada di BPSB Jawa Barat

Alat yang berada di BPSB Jawa Barat adalah germinator dengan sensor-sensor untuk menjaga lingkungan pembenihan. Sensor YL-90 adalah sensor untuk mengukur kelembaban media. Sensor ini diatur pada kelembaban 60%, sehingga saat kelembaban dibawah 60% maka sprinkler akan otomatis mengeluarkan air. Sensor GY-30 adalah sensor untuk mengukur intensitas cahaya. Sensor ini diatur pada intensitas 750lux, sehingga saat intensitas dibawah 750lx maka lampu akan otomatis. Sensor DHT-11 adalah sensor untuk mengukur suhu. Sensor ini diatur pada suhu 24-26°C, sehingga saat suhu berada dibawah 23°C maka heater akan otomatis menyala dan saat suhu berada diatas 25°C maka kipas akan otomatis menyala. Kamera ESP32 yang dipasang diatas germinator akan menangkap gambar sekali tiap hari.
